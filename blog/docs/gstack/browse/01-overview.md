# browse 架构与命令总览

> **状态**：导读骨架，欢迎按 PR 补函数级链路与图示。

## 职责

- 启动无头（或可配置有头）浏览器会话，暴露 **低延迟** 子命令：导航、选择器、截图、快照对比等。
- 与技能文档中的 **`$B`** 约定配合，避免在技能里硬编码 Playwright 细节。

## 阅读顺序建议

1. `browse/src/commands.ts` — 看清对外暴露的命令名与参数。
2. `browse/src/snapshot.ts` — 理解快照与标志位。
3. 按需深入单命令实现文件（以 `commands` 中的导入为准）。

## 构建产物

- 编译输出位于 `browse/dist/`（本地开发通过 `bun run build`；勿提交与平台绑定的二进制到本仓库——以仓库 **CLAUDE.md** 为准）。
